Scroll up/down
--------------

English:

	* :kbd:`page   down`
	* :kbd:`page   up`
	* :kbd:`pagedown`

macOS-ish:
	* :kbd:`⇧-⌘-r` / :kbd:`shift-cmd-r`
	* :kbd:`⌃-⌥-⌘` / :kbd:`ctrl-opt-r`


Insert non-breaking space
-------------------------

macOS:
	:kbd:`opt-space`
	
Windows:
	:kbd:`Alt+0160`
